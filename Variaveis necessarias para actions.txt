Ação "signup":
username: Nome de usuário para cadastro.
password: Senha do usuário. Esta será hasheada antes de ser salva no banco de dados.

Ação "login":
username: Nome de usuário para login.
password: Senha do usuário. Esta será verificada contra a hash armazenada.

Ação "send_message":
username: Nome de usuário que envia a mensagem.
message: Conteúdo da mensagem a ser enviada.
destination: Destinatário da mensagem.

Ação "create_group":
group_name: Nome do grupo a ser criado.
username: Nome de usuário que está criando o grupo.

Ação "add_to_group":
group_name: Nome do grupo para adicionar usuários.
username: Nome de usuário que está adicionando outros ao grupo.
member_usernames: Lista de nomes de usuários a serem adicionados ao grupo.

Ação "send_group_message":
username: Nome de usuário que envia a mensagem.
group_name: Nome do grupo para o qual a mensagem é destinada.
message: Conteúdo da mensagem a ser enviada.

Ação "delete_group":
group_name: Nome do grupo a ser deletado.
username: Nome de usuário que solicita a exclusão do grupo.
